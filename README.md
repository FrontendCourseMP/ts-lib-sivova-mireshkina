# Документация по вашему решению

## Описание

Учебная библиотека на TypeScript для работы с полями ввода. Библиотека позволяет задавать значения полей и проверять данные на валидность с учётом различных условий.

---

## Функциональные требования

- Библиотека должна выполнять проверку значений полей ввода на валидность.
- Библиотека должна принимать значение поля для проверки.
- Библиотека должна определять, является ли значение валидным или невалидным.
- Библиотека должна корректно обрабатывать пустые и некорректные значения.

---

## Нефункциональные требования

- Библиотека должна быть написана на TypeScript.
- В проекте должно быть реализовано модульное тестирование.
- Поведение библиотеки должно быть предсказуемым и воспроизводимым.

---

## Пример использования

```ts
import * as m from "./lib";

const check = m.form(document.querySelector("form"));
check.field("name").required("заполните поле").min("должно быть минимум 3");

check.validate(console.log);
```

Пример демонстрирует корректный и некорректный сценарии проверки значений

---

## API

```ts
form(element: HTMLFormElement | null)

Создаёт объект для валидации формы.

Параметры:

element (HTMLFormElement | null) — HTML-элемент формы, для которой настраивается валидация.

Возвращает:

объект валидатора формы.

field(name: string)

Добавляет поле формы для валидации.

Параметры:

name (string) — имя поля формы.

Возвращает:

объект поля с методами задания правил валидации.

required(message: string)

Добавляет правило обязательного заполнения поля.

Параметры:

message (string) — сообщение об ошибке при отсутствии значения.

min(message: string)

Добавляет правило минимальной длины значения поля.

Параметры:

message (string) — сообщение об ошибке при несоответствии правилу.

validate(callback: Function): void

Запускает проверку всех полей формы.

Параметры:

callback (Function) — функция, вызываемая с результатом проверки.

```
